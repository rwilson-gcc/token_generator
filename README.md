# ✅ **Token Generator**

A simple, lightweight, browser‑based **token generator** utility built using **HTML, CSS, and JavaScript**.  
This tool allows users to quickly generate random tokens for development, testing, and educational purposes—all locally in the browser with **no backend required**.

***

## 🚀 Features

*   ✅ 100% client‑side (no data sent over the internet)
*   ✅ Fast, simple token generation
*   ✅ Works offline
*   ✅ No external dependencies
*   ✅ Minimal, clean HTML interface

***

## 📂 Project Structure

    token_generator/
    │── index.html     # Main application UI and logic

## 🧪 How to Use

1.  Clone the repository:
    ```bash
    git clone https://github.com/rwilson-gcc/token_generator
    ```
2.  Open the project folder:
    ```bash
    cd token_generator
    ```

3.  Open `index.html` in any modern web browser.
